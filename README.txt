== Description ==

Small extension for the Twitter Tools plugin to exlude imported tweets in the feed.


== Installation ==

1. Extract the ZIP file and move the folder "twitter-tools-exclude-tweets-in-feed", with it contents, 
   to `/wp-content/plugins/` in your WordPress installation
2. Activate the pluing under 'Plugins' in the WordPress admin area
